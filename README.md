## 用法

### 一、克隆项目到本地

#### （1）git 克隆

```bash
git clone https://github.com/twgh/xcgui-example
```

#### （2）没有 git 的可以下载源码 zip 到本地后解压

[点击下载](https://codeload.github.com/twgh/xcgui-example/zip/refs/heads/main)

### 二、在项目根目录里执行命令

```bash
go mod tidy
cd SimpleWindow && go run .
```
